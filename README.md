# a.magno.me

A personal website hosted on GitHub Pages.

## Description

This is a static website serving as a personal landing page. It features a minimal design with corner-positioned navigation elements containing contact information and links to external resources.

## Structure

- `index.html` - Main landing page
- `style.css` - Stylesheet
- `script.js` - JavaScript functionality
- `/portfolio` - PDF portfolio viewer
- `/works` - Works section

## Technologies

- HTML5
- CSS3
- JavaScript
- Google Fonts (Space Grotesk)

## Hosting

The website is hosted on GitHub Pages and served at `a.magno.me` via custom domain configuration.
